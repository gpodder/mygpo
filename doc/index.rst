gpodder.net Documentation
=========================

`gpodder.net <https://gpodder.net>`_ is a podcasting webservice. It connects
your podcast players and lets you discover new and interesting podcasts.

This documentation is intended for developers. Please refer to the *API
Documentation* if you are developing podcast applications and want to integrate
functionality from gpodder.net. Please read the *Developer Documentation* if
you want to contribute to gpodder.net itself.
